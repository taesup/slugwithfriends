# slugwithfriends
Slugging with your buddy.

###Requirements:###
 * Python 2.7.6  
 * pip  
 * node  
 * MongoDB


##Sets up the MongoDB##
pip install -r modals/requirements.txt


##Seeding the Database ##
python modals/seed.py

## Running MongoDB##
python modals/run.py


## Starting Frontend Server ##
node fr/server
